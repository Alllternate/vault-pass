# Vault Pass

## Features

* Generate strong passwords

* Save passwords

* Password check

* Open Source

# Clone repository

## Clone with HTTPS

````
git clone https://github.com/Alllternate/vault-pass.git
````
## Clone with SSH

````
git@github.com:Alllternate/vault-pass.git
````

# How to use

## Install dependencies

For the project to run correctly you must install the project dependencies, you can do it with the following command.

````
pip install -r requirements.txt
````

## Generated Password

A length must be specified, for example 16, once specified the password is generated and automatically saved in a .pw file.

## Save Password

A password must be specified to save, once specified it is saved in a .pw file.

## Read Passwords

The passwords from the passwords.pw file will be read and printed in the terminal. If the file does not exist, an error will appear.

## Check Password

A password is specified, the program will read it and depending on the characters contained in the password, it will show a level of strength, the three levels are: Low, Medium, Strong.
